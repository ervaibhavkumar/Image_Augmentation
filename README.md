# Image_Augmentation

This Project contains the code to augment images so that if we have a small dataset of images then by using image augmenation more data can be generated .

### Techniques :-
1. Rescale Images
2. Random Noise
3. Grey Scale
4. Color Conversion
5. Rotate Image
6. Rescale Intensity
7. Gamma Correction
8. Horizontal Flip
9. Vertical Flip
10. Red Blur
11. Green Blur
12. Blue Blur
